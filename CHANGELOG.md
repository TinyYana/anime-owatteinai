# Changelog

## 0.1.1 - 2026-07-02

### Changed

- 降低管理後台的卡片化程度，將總覽、動畫編輯審核、身份權限、測試與 Discord 設定改成更扁平的區段、表格與分隔線列表。
- 針對會持續變長的審核紀錄、動畫編輯提案、觀看紀錄與審計紀錄加入分段顯示，避免一次塞滿整頁。
- 我的追番清單現在會直接顯示動畫的觀看入口，清單、緊湊與封面格檢視都能直接開啟外部連結。
- 放寬「看到 EP」集數輸入框並移除瀏覽器 number spinner，避免數字被擠掉。
- 新增動畫頁在搜尋前會先顯示社群正在追與資料庫已收錄的作品，讓空頁面也有可操作的入口。

### Maintenance

- 同步版本號到 `0.1.1`，並開始保留專案更新歷史。
